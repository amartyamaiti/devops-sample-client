Cheat Sheet for Docker commands
Sample Commands

Run a container with image nginx

docket run nginx

Run container with name nginxweb with iterractive tty with port 80 mapped to host 8090 port

docker run -it --name nginxweb -p 8090:80 nginx

Run container with name nginxweb with volume mapped from host D:/docker/nginx/ to path /usr/share/nginx/html & iterractive tty with port 80 mapped to host 8090 port

docker run -it -v D:/docker/nginx/:/usr/share/nginx/html --name nginxweb -p 8088:80 nginx

Run container with name tecmint-web with with port 80 mapped to host 8088 port volume mapped from host D:/docker/nginx/ to path /usr/local/apache2/htdocs/ having image httpd and tag 2.4

docker run -dit --name tecmint-web -p 8088:80 -v D:/docker/nginx/:/usr/local/apache2/htdocs/ httpd:2.4
